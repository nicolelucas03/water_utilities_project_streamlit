# Water Utilities Dashboard
An interactive, data-driven dashboard for analyzing water utility performance across multiple African countries, providing actionable insights into financial health, service delivery, and operational efficiency.

## Overview
The Water Utilities Dashboard is a comprehensive business intelligence tool designed to monitor and analyze the performance of water utility companies across Cameroon, Lesotho, Malawi, and Uganda. This project transforms raw operational data into actionable insights through interactive visualizations and KPI tracking.

## Features 
- Section Pages that outline 
    - Financial Performance
    - Service Delivery
    - Operations and Production 
    - Access
- AI Intergration for Downloading Reports and AI Bot
- User Authentication

## Setup and Installation

1. Clone the repository:
```bash
git clone [https://github.com/[username]/water_utilities_project_streamlit.git](https://github.com/nicolelucas03/water_utilities_project_streamlit.git)
cd Repo_Structures
```

2. Create and activate virtual environment:
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

4. Run the dashboard:
```bash
streamlit run app.py
```
